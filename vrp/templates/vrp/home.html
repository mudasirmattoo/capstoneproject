About This Project

This capstone project demonstrates the application of the Capacitated Vehicle Routing Problem (CVRP) using Google OR-Tools. It generates synthetic customer locations based on a chosen center point (via geocoding or browser location), retrieves real road distances using OpenRouteService, and calculates optimized routes respecting vehicle capacities. The results are visualized on an interactive map. Future enhancements could include incorporating machine learning for travel time prediction.
